# TIPI

(Pronounced tip-ee)

Turn a Raspberry PI and some glue hardware into a TI Disk Drive and Network interface. 

Keep it open for further device extension.

## Documentation

Please refer to the [wiki](https://github.com/jedimatt42/tipi/wiki)
